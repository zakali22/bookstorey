# Bookstorey web app
## Tech used:
* GatsbyJS
* Netlify Serverless functions
* Firebase - Authentication, Database and Storage

## Key features:
* Books, categories and authors listing
* Create an account, manage account details incl. profile image
* Add book favourites (guarded path)
* Dark mode
